# IAM

## What is IAM?
IAM (Identity and Access Management) is an AWS service used to manage access to AWS resources.

## Why is IAM used?
- To control who can access AWS resources
- To improve security
- To assign permissions

## Main Components
- Users
- Groups
- Roles
- Policies

## Key Point
Always follow the Principle of Least Privilege, meaning users should only get the permissions they need.
